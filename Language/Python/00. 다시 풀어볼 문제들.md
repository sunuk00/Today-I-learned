### 4번
아래 3개의 리스트 를 {key : a , value : b * c} 형태의  Dict(딕셔너리) 구조 로 변경하세요. *다양한 방법 활용*

아래 조건 참조
```python
a = ["one", "two", "three", "four"]
b = [30, 20, 15, 75]
c = [5.2, 7.4, 3.6, 4.2]
```
출력 결과 : `"{'one': 156.0, 'two': 148.0, 'three': 54.0, 'four': 315.0}"`

### 3번
```python
import this

d = {'a': 8, 'b': 33, 'c': 15, 'd': 26, 'e': 12, 'f': 120}

d2 = {}
for i in d:
    if d[i] >= 25:
        d2[i] = d[i]
print(d2)

d3 = {k : v for k, v in d.items() if v >= 25}
print(d3)

d4 = dict(filter(lambda x: x[1] >= 25, d.items())) # 🎈d.items()는 tuple를 반환함
print(d4)
```
