### 3번
아래와 같은 딕셔너리 구조에서 Value 값이 25 이상 필터링 후 출력하세요. 다양한 방법 으로 코딩하세요.

`d = {'a': 8, 'b': 33, 'c': 15, 'd': 26, 'e': 12, 'f': 120}`
출력 결과 : `{'b': 33, 'd': 26, 'f': 120}`

## Answers

### 1번
```python
# 코딩 영역
import this

x = ["grapes", "mango", "orange", "peach", "apple", "lime", "banana", "cherry", "tomato", "kiwi", "blueberry", "watermelon"]

# 방법 1.
# l = [i.upper() for i in x if i == "apple" or i == "kiwi"]
# print(f"{id(l)} and {id(x)}")

# 방법 2.
# l = list(map(lambda i : i.upper() , filter(lambda x : x == "apple" or x == "kiwi", x)))
l = list(map(str.upper, filter(lambda x : x == "apple" or x == "kiwi", x)))
print(l)
```

### 3번
```python
import this

d = {'a': 8, 'b': 33, 'c': 15, 'd': 26, 'e': 12, 'f': 120}

d2 = {}
for i in d:
    if d[i] >= 25:
        d2[i] = d[i]
print(d2)

d3 = {k : v for k, v in d.items() if v >= 25}
print(d3)

d4 = dict(filter(lambda x: x[1] >= 25, d.items()))
print(d4)
```
